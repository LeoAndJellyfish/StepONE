<div align="center">

## 特性

### 核心功能

- **成就管理** - 轻松创建、编辑、删除和查看您的各类成就
- **分类系统** - 6 种预定义分类，支持自定义分类管理
- **智能筛选** - 支持按分类、时间等多维度筛选成就
- **全文搜索** - 快速搜索成就标题和描述
- **标签系统** - 完全自定义标签，灵活标记成就

### 设计亮点

- **现代极简 UI** - 采用数字水彩画风格
- **毛玻璃效果** - 半透明玻璃拟态卡片
- **柔和配色** - 粉彩配色方案
- **流畅动画** - 优雅的过渡动画
- **响应式设计** - 适配多种屏幕尺寸

### 简历生成

- 集成 [Reactive Resume](https://github.com/amruthpillai/reactive-resume)（OnGoing）
- 一键生成专业简历
- 支持多种简历模板
- 可导出 PDF 格式


## 使用说明

### 快速开始

1. **首次使用**
   - 应用首次启动会自动初始化数据库
   - 系统会自动创建 6 个默认分类

2. **创建成就**
   - 点击主页右下角「+」按钮
   - 填写成就基本信息
   - 选择所属分类
   - 添加标签（可选）
   - 点击保存

3. **搜索成就**
   - 点击搜索图标或搜索区域
   - 输入关键词
   - 点击搜索按钮

4. **筛选成就**
   - 点击「全部」/「分类」/「最近」筛选器
   - 选择「分类」时可点击具体分类卡片

5. **编辑成就**
   - 点击成就卡片
   - 修改信息
   - 保存更新

### 最佳实践

- **及时记录**: 获得成就后及时录入系统
- **自定义标签**: 根据实际需求创建个性化标签体系
- **合理分类**: 选择合适的分类便于后续检索
- **定期整理**: 定期查看和整理成就记录
